Draggable

Options

cancel

Type
Selector
Default
input, textarea, button, select, option

Prevents dragging from starting on specified elements.

Example

{% highlight js %}$('#example-cancel').draggable({ cancel: '.title' }){% endhighlight %}
Title

clone

Type
Boolean
Default
false

Specifies whether the draggable should be cloned or not.

Example

{% highlight js %}$('.draggable-clone').draggable({ clone: true }){% endhighlight %}

cloneClass

Type
Boolean
Default
false

Specifies a class that will be added to cloned draggables.

Example

{% highlight js %}$('.draggable-cloneClass').draggable({ clone: true, cloneClass: 'clone' }){% endhighlight %}

connectWith

Type
Selector
Default
false

Allows the draggable to be dropped onto the specified sortables. If this option is used, a draggable can be dropped onto a sortable list and then becomes part of it.

Example

{% highlight js %}$('#list').sortable() .on('sortable:receive', function(e, ui) { ui.item.removeClass('draggable') }) $('#example-sortable .draggable').draggable({ connectWith: '#list' }){% endhighlight %}

disabled

Type
Boolean
Default
false

Disables the draggable if set to true.

Example

{% highlight js %}$('#example-disabled').draggable({ disabled: true }){% endhighlight %}

handle

Type
Selector
Default
false

If specified, restricts dragging from starting to the specified element(s). Only elements that descend from the draggable element are permitted.

Example

{% highlight js %}$('#example-handle').draggable({ handle: '.title' }){% endhighlight %}
Title

scope

Type
String
Default
"default"

Used to group sets of draggable and droppable items, in addition to the accept option. A draggable with the same scope value as a droppable will be accepted.

Example

{% highlight js %}$('#example-scope').draggable({ scope: 'T' }){% endhighlight %}
T

Methods

destroy()

Removes the draggable functionality completely.

{% highlight js %}$('#example-destroy').draggable() $('#example-destroy').draggable('destroy'){% endhighlight %}

disable()

Disables the draggable.

{% highlight js %}$('#example-disable').draggable('disable'){% endhighlight %}

enable()

Enables the draggable.

{% highlight js %}$('#example-enable').draggable('enable'){% endhighlight %}

option()

Gets an object containing key/value pairs representing the current draggable options hash.

{% highlight js %}var options = $('#example-option').draggable('option'){% endhighlight %}

option(optionName)

Gets the value currently associated with the specified optionName.

{% highlight js %}var isDisabled = $('#example-option').draggable('option', 'disabled'){% endhighlight %}

option(optionName, value)

Sets the value of the draggable option associated with the specified optionName.

{% highlight js %}$('#example-option').draggable('option', 'disabled', true){% endhighlight %}

option(options)

Sets one or more options for the draggable.

{% highlight js %}$('#example-option').draggable('option', { disabled: true }){% endhighlight %}

Events

draggable:create(event)

Triggered when the droppable is created.
